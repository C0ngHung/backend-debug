# Project Brief

## Overview
Backend service for debugging and training at SmartOSC. This project focuses on high-impact banking transaction bugs and best practices in Java/Spring Boot development.

## Core Requirements
- Professional Java 21 backend.
- Spring Boot architecture with clear separation of concerns.
- Robust transaction management for banking scenarios.
- Security-first approach with JWT.
- Integrated debugging demonstrations.

## Goals
- Serve as a learning platform for backend engineers.
- Document and demonstrate common production bugs (naming conflicts, ghost transactions, race conditions).
- Maintain high code quality and documentation standards.
